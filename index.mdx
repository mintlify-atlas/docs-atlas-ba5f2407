## Build modern headless storefronts

Tune Me In combines the power of Shopify Hydrogen with Sanity's structured content management to create fast, flexible e-commerce experiences. Build custom storefronts with editorial-rich product pages and seamless content integration.

## What's Inside

Tune Me In provides everything you need to build production-ready headless storefronts:

- **Product Listings**: Collection pages and product detail pages with variant support
- **Editorial Pages**: Rich content pages powered by Sanity's Portable Text
- **Cart Functionality**: Full shopping cart with Hydrogen's built-in state management
- **Custom Hooks**: Pre-built hooks for data fetching and context access
- **Responsive Components**: Mobile-first component library with Tailwind CSS
